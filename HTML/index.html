My First Project

My first major project was in Python, where I created a program called Random Restaurant Generator. The purpose of this program was to help users select a restaurant randomly without wasting time, and deciding where to go. The program had two buttons 'Generate now' and 'Generate again'. The 'Generate now' button randomly selected the name of a restaurant from a list of restaurants. If the user was not satisfied with the first selection, they had the option to 'Generate again'. After the name was randomly generated, the program also showed the menu and prices of the restaurant.
